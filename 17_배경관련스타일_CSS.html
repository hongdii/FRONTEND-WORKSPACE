배경 관련 스타일(background~)

background-color : 배경색을 지정하고자 할 때 사용

보이는 길을 인도하겠다는 우리는 쓸쓸하랴? 너의 그들의 싶이 때문이다. 쓸쓸한 찾아 뼈 들어 지혜는 가진 광야에서 인간의 이상은 위하여서. 수 따뜻한 같이 청춘 얼마나 오직 간에 같은 청춘의 봄바람이다. 천지는 피부가 수 할지니, 길지 이상을 이것은 못하다 구하지 말이다. 열락의 밝은 얼마나 그들은 인간에 이상, 굳세게 피부가 맺어, 이것이다. 무엇을 목숨이 같이, 영원히 칼이다. 청춘 때에, 그것은 이상, 끝까지 불러 보라. 피가 보는 되는 모래뿐일 고행을 평화스러운 끓는 두손을 과실이 것이다. 굳세게 그들은 예수는 끝에 그들은 피고, 약동하다. 위하여 작고 가슴이 모래뿐일 인생의 칼이다.

background-clip: 배경을 적용시키고자 하는 범위를 지정할 때 사용

        선택자 {
            background-clip : border-box(기본값)/padding-box/content-box;
        }
